## ⚠️ 潜在风险（需注意）

### 1. 文件访问 - 🟡 中等风险

**风险描述：**
UI 允许编辑工作区文件（`.md` 文件），如果 Token 泄露可能导致未授权修改。

**受影响路由：**
```
PUT/PATCH /api/files/content
Scope: memory | workspace
```

**当前保护：**
- ✅ 需要 Token 认证
- ✅ 仅限 `.md` / `.markdown` 扩展名
- ✅ 跳过敏感目录（`node_modules`, `.git`, `dist`）
- ✅ 文件大小限制（1MB）

**建议：**
1. 不要将 `LOCAL_API_TOKEN` 分享给他人
2. 定期更换 Token
3. 不要启用 `UI_BIND_ADDRESS=0.0.0.0`（除非必要）

---

### 2. 跨站请求伪造 (CSRF) - 🟡 低风险

**风险描述：**
UI 使用 Cookie/Session 认证时可能存在 CSRF 风险。

**当前状态：**
- ✅ 使用 Header 认证（`x-local-token`）
- ✅ 无 Cookie/Session 机制
- ✅ 仅本地访问（127.0.0.1）

**风险等级：** 🟡 低（本地访问 + Header 认证）

---

### 3. 信息泄露 - 🟡 低风险

**风险描述：**
UI 展示敏感信息（记忆、任务、会话内容）。

**当前保护：**
- ✅ 仅本地访问
- ✅ 需要 Token 认证才能修改
- ✅ 只读模式默认启用

**建议：**
1. 不要在公共屏幕上展示 UI
2. 使用后关闭浏览器
3. 定期清理敏感记忆
