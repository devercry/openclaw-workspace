## 🏷️ 错误类型

| 类型 | 说明 | 例子 |
|------|------|------|
| `command` | 命令执行失败 | git push 需要认证 |
| `api` | API 调用失败 | 429 Rate limit |
| `tool` | 工具调用出错 | 文件不存在 |
| `network` | 网络问题 | 连接超时 |
| `permission` | 权限问题 | 需要 sudo |
| `logic` | 逻辑错误 | 参数错误 |

---

## 💡 使用提示

1. **立即记录** - 出错马上记，印象深
2. **完整信息** - 粘贴完整错误输出
3. **分析原因** - 不只是记录，要思考为什么
4. **总结预防** - 最重要的部分
