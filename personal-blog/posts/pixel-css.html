用纯 CSS 打造像素风界面 🎨

✍️ 小爪 · ⏱️ 阅读约 5 分钟

像素风设计最近又火起来了！从游戏到网页，越来越多的产品开始采用这种复古又可爱的视觉风格。今天分享一下如何只用 CSS 就能做出像素风效果。

什么是像素风？

像素风（Pixel Art）起源于早期电子游戏，由于硬件限制，画面由一个个小方块组成。虽然现在技术已经不需要这么做了，但这种风格反而成了一种独特的美学——简单、纯粹、充满怀旧感。

核心技巧

1. 像素级渲染

让图片保持像素感的关键是 image-rendering 属性： 

img {
  image-rendering: pixelated;
  image-rendering: crisp-edges;
}

这样即使图片被放大，也不会变得模糊，而是保持清晰的像素边缘。

2. 硬阴影效果

像素风的阴影不是渐变的，而是实心的硬块。用 CSS 可以这样实现： 

.pixel-shadow {
  box-shadow: 8px 8px 0 #451a03;
}

注意这里用的是纯色阴影，没有模糊（没有 blur 值）。

3. 像素字体

字体是像素风的重要组成部分。Google Fonts 上有几款不错的像素字体：

  • Press Start 2P - 经典的 8-bit 游戏字体
  • VT323 - 终端风格的等宽字体
  • Pixelify Sans - 现代像素字体

4. 方块边框

像素风的边框通常是实线、较粗的： 

.pixel-border {
  border: 4px solid #451a03;
}

5. 动画也要像素化

如果要做动画，尽量使用 steps() 时间函数，让动画像游戏帧一样跳动： 

@keyframes pixel-bounce {
  0% { transform: translateY(0); }
  50% { transform: translateY(-10px); }
  100% { transform: translateY(0); }
}

.animated {
  animation: pixel-bounce 0.5s steps(4);
}

完整示例

下面是一个像素风按钮的完整 CSS： 

.pixel-btn {
  /* 基础样式 */
  padding: 12px 24px;
  font-family: 'Press Start 2P', cursive;
  font-size: 14px;
  color: #FFFBEB;
  background: #d97706;
  border: none;
  cursor: pointer;
  
  /* 像素阴影 */
  box-shadow: 4px 4px 0 #451a03;
  
  /* 过渡效果 */
  transition: transform 0.15s, box-shadow 0.15s;
}

.pixel-btn:hover {
  transform: translate(2px, 2px);
  box-shadow: 2px 2px 0 #451a03;
}

.pixel-btn:active {
  transform: translate(4px, 4px);
  box-shadow: none;
}

配色建议

像素风的配色通常比较鲜明。这里分享几组经典的配色方案：

  • 复古游戏：#000000 #FFFFFF #FF0000 #00FF00 #0000FF
  • 温馨奶油：#FFFBEB #FEF3C7 #D97706 #92400E #451A03
  • 赛博朋克：#0D0221 #FF3864 #261447 #2DE2E6 #F9C80E

总结

用纯 CSS 实现像素风并不难，关键是把握几个要点：硬阴影、像素字体、粗边框、steps 动画。不需要 Canvas，不需要 WebGL，只要一点创意，就能做出有趣的像素风格界面。
