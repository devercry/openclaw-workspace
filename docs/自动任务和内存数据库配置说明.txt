│  ⚠️  需要配置：Embedding Provider API Key              │

内存搜索需要 embedding provider 才能工作。

选择以下任一方案：

方案 1: OpenAI（推荐）
───────────────────────────────────────────────────────────
  export OPENAI_API_KEY="你的 key"
  
  添加到 ~/.bashrc：
  echo 'export OPENAI_API_KEY="sk-xxx"' >> ~/.bashrc
  source ~/.bashrc

方案 2: Google Gemini
───────────────────────────────────────────────────────────
  export GEMINI_API_KEY="你的 key"
  
  然后修改 ~/.openclaw/openclaw.json：
  "memorySearch": {
    "enabled": true,
    "provider": "google"
  }

方案 3: Voyage AI
───────────────────────────────────────────────────────────
  export VOYAGE_API_KEY="你的 key"
  
  然后修改 ~/.openclaw/openclaw.json：
  "memorySearch": {
    "enabled": true,
    "provider": "voyage"
  }

方案 4: 本地模型（无需 API Key，但需要下载模型）
───────────────────────────────────────────────────────────
  配置本地 embedding 模型路径
