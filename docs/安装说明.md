## ⚠️ 注意事项

### 1. 只读模式（默认）

**安全配置：**
```bash
SECURITY_READONLY=true
```

**影响：**
- ✅ 可以查看状态、日志、记忆
- ❌ 不能修改配置、删除数据
- ❌ 不能执行危险操作

**如需编辑：**
```bash
# 编辑 .env
SECURITY_READONLY=false

# 重启
npm run dev:ui
```

---

### 2. 仅本地访问

**默认绑定：**
```bash
UI_HOST=127.0.0.1
```

**如需远程访问：**
```bash
# 编辑 .env
UI_HOST=0.0.0.0

# ⚠️ 注意安全，建议配合认证
```

---

### 3. 需要 Gateway 运行

**依赖：**
- OpenClaw Gateway 必须运行
- WebSocket 连接：`ws://127.0.0.1:18789`

**检查：**
```bash
openclaw gateway status
```
