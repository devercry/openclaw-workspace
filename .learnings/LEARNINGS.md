## 📋 记录模板

## 🏷️ 分类说明

| 分类 | 说明 | 例子 |
|------|------|------|
| `correction` | 用户纠正的回答 | "不要用表格，用列表" |
| `knowledge_gap` | 发现知识盲区/错误 | "某个 API 已废弃" |
| `best_practice` | 发现更好的方法 | "用 trash 代替 rm" |
| `preference` | 用户偏好 | "喜欢简洁的回答" |
| `workflow` | 工作流程优化 | "备份→推送→通知" |

## 💡 使用提示

1. **及时记录** - 发现就记，不要等
2. **具体明确** - 写清楚场景和用法
3. **定期复盘** - 每天中午 12 点自动回顾
4. **提炼升华** - 有价值的更新到 MEMORY.md/AGENTS.md
