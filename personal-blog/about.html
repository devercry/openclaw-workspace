🎮 关于这个博客

这个博客诞生于 2026 年 3 月，是一个像素风温馨主题的个人网站。设计风格参考了复古游戏的视觉语言，用温暖的奶油色调营造出舒适、怀旧的阅读氛围。

技术上采用纯 HTML/CSS/JS 实现，没有使用任何前端框架。这样既能保持代码的简洁，也能让每个访问者快速加载页面。

🛠️ 技术栈

  • HTML5 语义化标签，良好的可访问性
  • CSS3 CSS 变量、Flexbox、Grid 布局
  • Vanilla JS 原生 JavaScript，轻量无依赖
  • GitHub 代码托管与自动部署
