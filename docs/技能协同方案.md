## 🔗 协同工作流程

### 流程 1：自动备份 + 通知
```
用户触发 → github 备份 → qqbot-remind 发送通知
```

**触发词：**
- "备份代码"
- "提交更改"
- "推送到 GitHub"

**涉及技能：**
1. github（主）
2. qqbot-remind（辅）

---

### 流程 2：内容创作工作流
```
用户说要写文章 → tavily 搜索素材 → 创作内容 → humanizer-zh 去 AI 味 → qqbot-media 发送
```

**触发词：**
- "写一篇文章"
- "创作内容"
- "帮我写"

**涉及技能：**
1. tavily（素材收集）
2. humanizer-zh（内容优化）
3. qqbot-media（结果发送）

---

### 流程 3：新技能安装流程
```
用户说要找技能 → find-skills 发现 → skill-vetter 安全审查 → 用户确认 → 安装 → self-improvement 记录
```

**触发词：**
- "找技能"
- "安装技能"
- "有什么技能"

**涉及技能：**
1. find-skills（发现）
2. skill-vetter（审查）
3. skillhub-preference（安装）
4. self-improvement（学习记录）

---

### 流程 4：系统健康检查
```
定时触发 → healthcheck 检查 → node-connect 诊断 → qqbot-remind 报告结果
```

**触发词：**
- "检查系统"
- "安全审计"
- "健康检查"

**涉及技能：**
1. healthcheck（主）
2. node-connect（辅）
3. qqbot-remind（报告）

---

### 流程 5：视频处理工作流
```
用户提供视频 → video-frames 截图/剪辑 → qqbot-media 发送结果
```

**触发词：**
- "截图"
- "剪辑视频"
- "提取帧"

**涉及技能：**
1. video-frames（主）
2. qqbot-media（发送）

---

### 流程 6：外部 API 调用
```
用户需要外部服务 → api-gateway 连接 → 处理结果 → 返回用户
```

**触发词：**
- "查询 Gmail"
- "查看日历"
- "操作 Google"

**涉及技能：**
1. api-gateway（主）
2. tavily（可选，补充搜索）

---

### 流程 7：工作流管理
```
复杂任务 → clawflow 分解 → 多个子任务并行 → 汇总结果
```

**触发词：**
- "执行工作流"
- "多步骤任务"
- "批量处理"

**涉及技能：**
1. clawflow（主）
2. 其他相关技能（按需调用）

---

### 流程 8：收件箱分类处理
```
收到消息 → clawflow-inbox-triage 分类 → 紧急的通知，其他的汇总
```

**触发词：**
- 自动触发（心跳/定时）

**涉及技能：**
1. clawflow-inbox-triage（主）
2. qqbot-remind（紧急通知）

## 📋 技能依赖关系

```
self-improvement
    └── 记录所有技能的使用情况

skill-vetter
    └── 审查新安装的技能

find-skills + skillhub-preference
    └── 协同发现和安装技能

clawflow
    └── 可以调用任何其他技能

qqbot-*
    └── 作为输出通道，配合所有技能
```

## 🎯 技能优先级规则

### 最高优先级（用户明确指定）
用户说"用 github 备份"，直接用 github，不经过匹配。

### 高优先级（匹配度最高）
用户说"备份代码"，匹配到 github（最高）> clawflow（中）。

### 中优先级（最近使用）
多个技能匹配度相同时，选择最近使用的。

### 低优先级（默认）
没有明确匹配时使用默认技能。

## ⚠️ 冲突处理

### 场景 1：多个技能都能处理
**用户说：** "搜索一下"

**可能匹配：**
- tavily（网络搜索）
- api-gateway（特定 API 搜索）

**处理方式：**
询问用户："想用 Tavily 搜索网络，还是用 API Gateway 查询特定服务？"

---

### 场景 2：技能依赖缺失
**用户说：** "写文章并发送"

**需要：**
- humanizer-zh（内容优化）
- qqbot-media（发送）

**如果 qqbot-media 缺失：**
提示："需要 qqbot-media 技能才能发送消息，要安装吗？"
